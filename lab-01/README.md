# Lab 1: Hands-on Cloud Security with FortiCNAPP

## Objectives

- Understand FortiCNAPP architecture and capabilities
- Navigate the FortiCNAPP console
- Review key security features and dashboards

## Prerequisites

- FortiCNAPP account access
- Web browser

## Lab Steps

### Step 1: Access FortiCNAPP Console

1. Navigate to the FortiCNAPP console
2. Log in with your credentials
3. Review the main dashboard

### Step 2: Explore Security Features

1. Review compliance dashboards
2. Examine threat detection capabilities
3. Navigate through inventory views

### Step 3: Review Integration Options

1. Explore available cloud integrations
2. Review agent-based and agentless options
3. Understand data collection methods

## Expected Results

- Familiarity with FortiCNAPP interface
- Understanding of core security capabilities
- Knowledge of integration approaches
