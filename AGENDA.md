# FortiCNAPP Technical Enablement Workshop Agenda

## Overview

Hands-on labs for FortiCNAPP (Fortinet Cloud Native Application Protection Platform) AWS cloud security integration.

## Lab Schedule

- [Lab 1: Hands-on Cloud Security with FortiCNAPP](lab-01/README.md)
  - Introduction to FortiCNAPP console and capabilities
  - Navigation and dashboard overview

- [Lab 2: Install Lacework CLI and Configure](lab-02/README.md)
  - CLI installation and configuration
  - API connectivity verification

- [Lab 3: Install Terraform](lab-03/README.md)
  - Terraform installation
  - AWS provider configuration

- [Lab 4: Integrate Inventory and Audit (AWS - Configuration and CloudTrail)](lab-04/README.md)
  - AWS CloudTrail integration
  - AWS Config integration
  - Inventory and audit data collection

- [Lab 5: Integrate Agentless Workload Scanning](lab-05/README.md)
  - Agentless scanning configuration
  - Vulnerability scanning setup
  - Scan results review

- [Lab 6: Install Linux Agent](lab-06/README.md)
  - Linux agent installation
  - Agent configuration and verification
  - Data collection validation

- [Lab 7: Install Windows Agent](lab-07/README.md)
  - Windows agent installation
  - Windows-specific monitoring
  - Security data collection

## Workshop Duration

Estimated total time: 4-6 hours

## Prerequisites

- AWS account with administrative access
- FortiCNAPP account access
- Command-line terminal access
- Basic knowledge of AWS services
